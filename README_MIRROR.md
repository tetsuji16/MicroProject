GitHub ミラー計画（CPAL 1.0）
------------------------------------------------------------
このリポジトリは、SourceForge 上の ProjectLibre のコードを GitHub にミラーすることを目的とします。現在は READ-WRITE での実装準備段階です。

目的
- CPAL 1.0 のライセンス通知を遵守しつつ、ミラーとして安定運用できる状態へ整備する。
- 将来的には Rust + TAURI 版の書き直し計画と連携させる基盤を整える。

範囲
- SourceForge 由来のコードベースのミラー化（履歴の保持方針は初期案として完全履歴を想定）。
- ライセンス通知のファイル（LICENSE、NOTICE など）の取り扱いを明記。
- 初期 Readme、CONTRIBUTING、ライセンス表記のドラフトを用意。

運用方針
- 同期頻度、ブランチ運用、アップデート反映のルールは後続で決定。
- 将来の Rust + TAURI 移植と連携するためのドキュメントを並行して整備する。

将来計画
- 完全履歴ミラーの運用を継続。移植計画と整合性を図る。
- 法務・ライセンス周りの確認を適宜実施。

注意事項
- 実際のリポジトリ作成・マージ・プッシュ等の操作は、あなたの承認と指示を待ちます。
